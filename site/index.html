// how it’s made

Built in the open, with CodeYam.

Otterpace is open source, and it was built with CodeYam — a way to build apps as living, testable scenarios. Every screen here, including all five themes, is a real captured scenario: real data, real states, screenshotted exactly as they render. It’s how the whole app stays verifiable — and how you can read every line yourself.

01

Build screens as scenarios

Each state — a crushed goal, a quiet day, every theme — is defined with real data, not lorem ipsum.

02

Preview & capture live

Every scenario renders in a real simulator and is screenshotted — the exact shots on this page.

03

Ship with tests, in the open

Logic is unit-tested, screens are visually covered, and the whole repo is public.
